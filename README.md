# 📄 Resume Scoring System

An AI-based web application that analyzes resumes and provides a score along with feedback for improvement.

---

## ✨ Features

* Upload resume files
* Resume scoring system
* Feedback suggestions
* Clean and simple interface

---

## 🛠️ Tech Stack

* Python (Flask)
* HTML
* Logic-based AI scoring

---

## 🚀 How to Run

1. Install Python

2. Install Flask:
   pip install flask

3. Run the application:
   python app.py

4. Open in browser:
   http://127.0.0.1:5000
